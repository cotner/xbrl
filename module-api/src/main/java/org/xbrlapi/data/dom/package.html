Implements the XBRL API data store using in-memory XML DOM objects.

This package implements the DBConnection and Store interfaces using in-memory XML DOM objects as the underlying data representation.

Package Specification

This package is intended to provide a very lightweight, fast, and simple datastore for the XBRLAPI. It is well suited to small amounts of XBRL data but scales badly as the size of the data store increases due to the need to keep the entire data set in memory.
