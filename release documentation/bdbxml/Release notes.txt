Release details:
The Oracle Berkeley Database XML data store implementation
for XBRLAPI.

Release 3.4
December 10, 2008
Fixed problems in construction of JAR files with the 3.3 release.

Release 3.3
November 24, 2008
Point release involving minor bug fixes.

Release 3.2
Jul 30, 2008
Point release to upgrade data store performance via improved
indexing and improved memory management.

Release 3.1
Feb 28, 2008
Point release to include various bug-fixes. 

Release 3.0
July 29, 2006
Created the bdbxml module to enable the Oracle Berkeley 
database to be used as a persistent data store for
XBRLAPI.  The version number is 3.0 to keep the module 
version numbers synchronised across the XBRLAPI distribution.
