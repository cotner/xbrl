The homepage for xbrlapi.org

XBRLAPI.ORG is dedicated to the definition and Java implementation of an opensource XBRL API. Sourceforge.net kindly provide the project hosting for XBRLAPI.ORG. The API specification and the Java implementation are released under the GPL license.

XBRLAPI.ORG is supported by Galexy. If your organisation also wishes to support the work programme of XBRLAPI.ORG then please consider contributing expertise and/or time. Most valuable to XBRLAPI.ORG would be a review of the API specification and contributions of development effort. Direct funding is not required.

The current API specification can be reviewed as a set of JavaDocs. These are currently extremely unstable.

The Java implementation strategy is also explained in some detail on this website. If you have constructive feedback on that strategy it would be most welcome.

This site is under construction.
