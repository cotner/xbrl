Tests the XLink processor and handler.

Package Specification

This package tests the functionality of the XLink processor.

Testing is done using a set of specific XLink handlers and a range of simple XLink XML structures hardcoded as strings in the tests themselves.

Testing is used to demonstrate that:

  • only true XLink elements are recognised
  • custom links are recognised
  • XLink attributes are correctly passed to handlers

xbrlapi.org Java library providing XBRL functionality
